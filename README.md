# Voyage — Holiday Planner

A holiday planning and itinerary web app with costings, budgeting, and smart travel-time estimates.

## Features

- **Trip dates** — Set your holiday start and end dates
- **Stops** — Add destinations with arrival and departure dates
- **Flights** — Full detail including airports, duration, connections, ticket cost, and recommended departure time based on distance, traffic, and personal buffer settings
- **Accommodation** — Check-in/out times, nightly rates, booking references
- **Rental cars** — Pick-up and drop-off with daily rates
- **Sightseeing** — Dates, times, ticket prices, opening hours, and file attachments for bookings (PDF, images)
- **Transport** — Trains, taxis, buses with distance-based travel time estimates
- **Chronological itinerary** — Summary line items grouped by date; click to expand full details
- **Budget summary** — Total and per-category cost breakdown
- **Time buffers** — Adjustable airport arrival buffer, general comfort padding, and traffic multiplier

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser. Data is saved automatically to local storage.

## Build

```bash
npm run build
npm run preview
```
